# 📄 محلل العقود الذكي

تطبيق ذكاء اصطناعي لتحليل عقود PDF بالعربي والإنجليزي.

---

## 🚀 تشغيل التطبيق

### 1. تثبيت المكتبات
```bash
pip install -r requirements.txt
```

### 2. إعداد API Key
```bash
# انسخ ملف الإعدادات
cp .env.example .env

# افتح .env وأضف مفتاحك
ANTHROPIC_API_KEY=sk-ant-xxxxxxxxx
```

### 3. تشغيل التطبيق
```bash
streamlit run app.py
```

سيفتح المتصفح تلقائياً على: http://localhost:8501
